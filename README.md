# MeatQuiz — установка на Android через GitHub Pages

Это устанавливаемое веб-приложение (PWA) — викторина «угадай отруб говядины по
фотографии». После публикации на GitHub Pages его можно открыть по ссылке и
установить на телефон как обычное приложение: своя иконка, на весь экран,
работает офлайн, прогресс сохраняется.

Как играть: на экране — фото отруба (если фото несколько, листаются свайпом)
и четыре варианта названия. Выбери правильный; в конце теста — отчёт с
подсказками и комментариями о том, как готовится каждый отруб.

## Что внутри
- `index.html` — само приложение
- `manifest.webmanifest` — описание приложения (имя, иконки)
- `sw.js` — офлайн-режим (service worker)
- `supabase.js` — библиотека для облачной синхронизации и рейтинга
- `icon-192.png`, `icon-512.png`, `icon-512-maskable.png` — иконки (буквы M Q)
- `dictionary/MEAT_QUIZ.js` — база вопросов (88 отрубов)
- `steak_images/` — фотографии отрубов: `<ID>_001.jpg`, `<ID>_002.jpg`, …
  (имя файла = ID вопроса; расширения .jpg/.png/.webp подбираются сами)
- `.nojekyll` — чтобы GitHub Pages отдавал файлы как есть

## Шаги (через сайт GitHub, без программ)

1. Зайди на https://github.com и войди/зарегистрируйся (бесплатно).
2. Справа вверху «+» → **New repository**.
   - Repository name: например `meatquiz`
   - Поставь **Public**
   - Нажми **Create repository**.
3. На странице репозитория: **Add file** → **Upload files**.
   - Перетащи СЮДА ВСЕ файлы из этой папки (включая `.nojekyll`,
     папку `dictionary` и папку `steak_images`).
   - Внизу нажми **Commit changes**.
4. Вкладка **Settings** (вверху) → слева **Pages**.
   - Source: **Deploy from a branch**
   - Branch: **main**, папка **/ (root)** → **Save**.
5. Подожди 1–2 минуты, обнови страницу Pages — появится ссылка вида
   `https://ТВОЙ-ЛОГИН.github.io/meatquiz/`

Важно: GitHub Pages чувствителен к регистру. Папка должна называться именно
`steak_images`, а файлы — совпадать с ID вопроса, например
`SIRLOIN_FILLET_001.jpg`.

## Установка на телефон (Android)
1. Открой эту ссылку в **Chrome** на телефоне.
2. Меню (⋮ вверху справа) → **Установить приложение**
   (или «Добавить на главный экран»).
3. Готово — на рабочем столе появится иконка **MeatQuiz** (буквы M Q).

Прогресс хранится в приложении на телефоне. Чтобы обновить приложение —
просто залей новый `index.html` (и `sw.js`) в репозиторий
(Add file → Upload files), телефон подтянет обновление при следующем запуске онлайн.

## Облако и рейтинг (Supabase)
Синхронизация прогресса и рейтинг работают через Supabase. Для нового
приложения нужна СВОЯ база: создай проект на https://supabase.com, выполни в
SQL Editor скрипт `meatquiz_supabase.sql` (лежит рядом с этим файлом) и
пропиши свои `SUPABASE_URL` и `SUPABASE_KEY` в `index.html`
(поиск по строке `SUPABASE_URL`). Подробная инструкция — внутри SQL-файла.

## Хочешь именно файл .apk?
Когда ссылка с шага 5 уже работает:
- Зайди на https://www.pwabuilder.com, вставь ссылку → раздел **Android**
  → **Generate Package** → скачаешь zip с `.apk`.
- На телефоне разреши установку из неизвестных источников и открой `.apk`.
